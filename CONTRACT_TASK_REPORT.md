## 🔍 开仓信号逻辑（AND组合，非评分）

### 🟢 做多（必须同时满足4个条件）：

| # | 条件 | 说明 |
|---|------|------|
| ① | RSI正背离 | 价格新低但RSI没有新低 |
| ② | MACD>0 | 柱状线翻红 |
| ③ | 布林中轨支撑 | 价格偏离中轨 < 2% |
| ④ | 附加（至少1个） | ADX>25+DI+排列 / vol_ratio>1.2 / RSI<50 |

### 🔴 做空（必须同时满足3个条件）：

| # | 条件 | 说明 |
|---|------|------|
| ① | RSI负背离 | 价格新高但RSI没有新高 |
| ② | 非强趋势 | ADX≤60 或 DI->DI+ |
| ③ | 附加（至少1个） | MACD<0 / 量价背离(vol<0.5) / 触及布林上轨 / 动量衰竭 / 1H涨幅>10% |

---

## ⚠️ 强趋势拦截规则

- ADX > 60 且 DI+ > DI- 时：禁止做空
- 强趋势中可以等待，但不开仓

## 💰 开仓规则

| 参数 | 值 |
|------|-----|
| 保证金 | 2 USDT/仓 |
| 杠杆 | 10x 逐仓 |
| 开仓数量 | 2÷价格×10 |
| 平仓方式 | 市价单 |
| 冷却时间 | 同一币种 > 5分钟（持久化） |

---

## 📈 移动止盈

| 参数 | 值 |
|------|-----|
| 触发门槛 | 价格偏离开仓价 ±6% |
| 追踪方式 | 多单记最高价，空单记最低价 |
| 退出门槛 | 从峰值反向偏离 4% |
| 止损 | 不设（纯移动止盈） |

---

## 🛡️ 风控机制

- 每仓最大亏损 = 2 USDT
- 逐仓隔离，仓位独立
- 冷却时间持久化（进程重启不丢失）
- 手动平仓检测（每5秒查持仓数量，发现已平则通知并清除记录）
- DB1涨幅<10%自动出库
