# 📌 第二章：现货仓位计算

## 2.1 现货仓位公式

```
买入数量 = 账户净值 × 仓位比例% ÷ 当前价格
```

## 2.2 各标的现货仓位上限

| 类别 | 标的示例 | 仓位上限 | 止损 |
|-----|---------|---------|------|
| 🥇 贵金属 | XAUUSD, XAGUSD | 10% | 5% |
| 📈 美股A+级 | TSLA, NVDA, COIN, MARA, RIOT | 5% | 8% |
| 📈 美股A级 | AMD, ARM, PLTR, SNOW, QBTS | 8% | 6% |
| 📈 美股B/C级 | AAPL, META, AMZN, GOOGL, KO, CVX | 10% | 5% |
| 📈 中概股 | BIDU, PDD, GRAB | 5% | 8% |
| 🥈 商品ETF | GLD, USO, COPX, REMX | 10% | 5% |
| 📊 指数ETF | TQQQ, SQQQ, VTI | 10% | 8% |
| 💵 债券ETF | SGOV | 20% | 无需止损 |

## 2.3 总仓位限制

```
最大持仓：账户净值 80%
最低现金：保留 20%

例如：账户1000U
→ 最多持仓 800U 的币
→ 保留 200U 现金
```

| SGOV | 无需止损 | 稳定收益 |

## 4.2 跟踪止损（现货可适当收紧）

```
入场后：
  - 上涨 3% → 止损移至 +1% 或保本
  - 上涨 5% → 止损移至 +3%
  - 上涨 8% → 止损移至 +5%
  - 持续上涨 → 跟踪最高点回落 5%
```

## 4.3 止盈规则

### RSI 极端值止盈
```
RSI > 75  → 减仓 1/3
RSI > 80  → 再减 1/3
RSI > 85  → 清仓
```

### 分批止盈
```
目标1：涨幅 8% → 减仓 1/3
目标2：涨幅 15% → 再减 1/3
目标3：涨幅 25% → 全部止盈
（或跟踪止损：最高点回落 8%）
```

# 📌 第五章：每日操作流程

## 盘前（09:00-09:30）

```
1. 查看期货/贵金属预判今日方向
2. 检查隔夜持仓状态
3. 计算今日可承受亏损额度
4. 列出今日关注标的
```

## 盘中（09:30-16:00）

```
每30分钟扫描：
□ 持仓是否触发止损？
□ 持仓是否触发止盈？
□ 有无新标的出现入场信号？
□ 需不需要加仓？（盈利才加）
□ 风控状态是否正常？
```

## 盘后（16:00-16:30）

```
1. 复盘今日所有交易
2. 记录盈亏和原因
3. 更新持仓清单
4. 准备明日交易计划
```

# 📌 第六章：风控规则

## 5.1 仓位上限

```
总持仓 ≤ 80% 账户净值
单标的 ≤ 按等级表执行
同类型标的 ≤ 30% 总资金
```

## 5.2 亏损熔断

```
🟢 正常：单笔亏损 2% 以内
🟡 警戒：单日亏损 3% → 禁止新开单
🔴 熔断：月度亏损 10% → 3天冷静期
```

## 5.3 连续亏损

```
亏损 2 次 → 仓位减半
亏损 3 次 → 检查策略
亏损 5 次 → 当日停止
```
