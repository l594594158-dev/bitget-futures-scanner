# 现货买卖信号策略手册

## 更新时间：2026-04-06

# 一、核心指标说明

## 1. 均线 (MA)

| 均线 | 周期 | 用途 |
|-----|------|------|
| MA5 | 5日 | 短期趋势 |
| MA10 | 10日 | 快速趋势 |
| MA20 | 20日 | 中期趋势 |
| MA50 | 50日 | 长期趋势 |
| MA200 | 200日 | 牛熊分界线 |

**信号：**
- MA5 > MA20 → 多头排列（看涨）
- MA5 < MA20 → 空头排列（看跌）

---

## 2. RSI 相对强弱指数

| RSI值 | 信号 | 含义 |
|-------|------|------|
| < 30 | 超卖 | 可能要反弹 |
| 30-50 | 偏弱 | 观望 |
| 50-70 | 偏强 | 可持有 |
| > 70 | 超买 | 可能要回调 |

---

## 3. MACD 指数平滑异同移动平均线

| 组成 | 说明 |
|-----|------|
| DIF线 | 快线（12日EMA - 26日EMA）|
| DEA线 | 慢线（DIF的9日EMA）|
| MACD柱 | DIF - DEA |

**信号：**
- MACD柱从负转正 → 金叉（买）
- MACD柱从正转负 → 死叉（卖）

---

## 4. 布林带 (Bollinger Bands)

| 轨道 | 含义 |
|-----|------|
| 上轨 | 压力位（超买区域）|
| 中轨 | 20日均线 |
| 下轨 | 支撑位（超卖区域）|

**信号：**
- 价格跌破下轨 → 超卖，可能反弹
- 价格突破上轨 → 超买，可能回调

---

# 二、进场（买入）信号

## 2.1 进场条件

**当同时满足以下5个条件时，可以考虑进场：**

| # | 条件 | 具体标准 |
|---|------|---------|
| 1 | 趋势确认 | MA5 > MA20（多头排列） |
| 2 | RSI位置 | RSI < 50（不是超买区域）|
| 3 | MACD信号 | MACD柱 ≥ 0（金叉或已经为正）|
| 4 | 支撑位 | 价格在MA20或布林中轨附近 |
| 5 | K线形态 | 出现阳线吞噬或锤子线 |

---

## 2.2 买入信号强度

| 满足条件数 | 信号强度 | 建议仓位 |
|-----------|---------|---------|
| 3个 | 弱 | 轻仓（10%）|
| 4个 | 中 | 半仓（25%）|
| 5个 | 强 | 正常仓（40%）|

---

## 2.3 买入信号列表

| 信号 | 说明 | 优先级 |
|-----|------|-------|
| 均线金叉 | MA5从下穿越MA20 | ⭐⭐⭐ |
| RSI超卖反弹 | RSI从<30回升 | ⭐⭐⭐ |
| MACD底部金叉 | MACD柱在0轴下方转正 | ⭐⭐⭐ |
| 布林下轨支撑 | 价格触及布林下轨后反弹 | ⭐⭐ |
| 缩量回调 | 回调时成交量萎缩 | ⭐⭐ |
| 底部吞没 | K线出现阳包阴 | ⭐⭐ |
| 放量突破 | 突破关键阻力位 | ⭐ |

---

# 三、离场（卖出）信号

## 3.1 离场条件

**当满足以下任一条件时，必须离场：**

| # | 条件 | 具体标准 |
|---|------|---------|
| 1 | 止损触发 | 亏损达到5% |
| 2 | RSI超买 | RSI > 75 |
| 3 | MACD死叉 | MACD柱从正转负 |
| 4 | 均线死叉 | MA5从下穿越MA20 |
| 5 | 跟踪止损 | 从最高点回落8% |

---

## 3.2 卖出信号列表

| 信号 | 说明 | 优先级 |
|-----|------|-------|
| 均线死叉 | MA5从上穿越MA20 | ⭐⭐⭐ |
| RSI超买 | RSI > 70 | ⭐⭐⭐ |
| MACD顶部死叉 | MACD柱在0轴上方转负 | ⭐⭐⭐ |
| 布林上轨压制 | 价格触及布林上轨后回落 | ⭐⭐ |
| 放量滞涨 | 涨时放量但价格不涨 | ⭐⭐ |
| 顶部吞没 | K线出现阴包阳 | ⭐⭐ |
| 高点回落 | 从近期高点回落5%以上 | ⭐ |

---

## 3.3 分批止盈策略

| 信号 | 动作 | 剩余仓位 |
|-----|------|---------|
| RSI > 70 | 减仓1/3 | 66% |
| RSI > 75 | 再减1/3 | 33% |
| RSI > 80 | 清仓 | 0% |

# 四、止损止盈规则

## 4.1 止损规则（强制执行）

| 类型 | 幅度 | 说明 |
|-----|------|------|
| 固定止损 | -5% | 入场价 × 0.95 |
| 时间止损 | 5天 | 5天内未盈利考虑出局 |
| 跟踪止损 | -3% | 从最高点回落3% |

---

## 4.2 止盈规则

| 目标 | 幅度 | 说明 |
|-----|------|------|
| 第一目标 | +8% | 减仓1/3 |
| 第二目标 | +15% | 再减1/3 |
| 第三目标 | +25% | 持有到死叉 |

---

## 4.3 仓位管理

| 信号强度 | 仓位比例 | 最大亏损 |
|---------|---------|---------|
| 强 | 40% | 总资金2% |
| 中 | 25% | 总资金1.25% |
| 弱 | 10% | 总资金0.5% |

**总仓位上限：不超过总资金60%**

# 五、实战流程

## 5.1 每日检查清单

```
□ 检查持仓标的是否触发止损
□ 检查持仓标的是否触发止盈
□ 扫描潜在买入标的
□ 更新市场状态
```

## 5.2 买入流程

```
1. 发现标的满足买入条件
2. 确认信号强度
3. 计算仓位和止损价
4. 下单买入
5. 设置止损单
6. 记录交易
```

## 5.3 卖出流程

```
1. 发现卖出信号
2. 判断是止盈还是止损
3. 执行对应卖出
4. 记录交易结果
5. 更新持仓
```

# 七、注意事项

## 7.1 不要做的事

```
❌ RSI超买时追高（RSI > 70）
❌ 均线空头时买入（MA5 < MA20）
❌ MACD死叉时不卖出
❌ 亏损超过5%不止损
❌ 重仓单一标的（仓位 > 40%）
```

## 7.2 一定要做的事

```
✅ 每次入场必须带止损
✅ 分批止盈，不贪心
✅ 记录每笔交易
✅ 亏损后减仓
✅ 连续亏损3次休息1天
```
