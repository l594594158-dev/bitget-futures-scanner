## 五、交易逻辑

### 5.1 开仓流程

```
触发信号 →
  获取当前价格 →
  计算数量 = (2 USDT × 10杠杆) ÷ 当前价格 →
  set_leverage(10x, isolated) →
  市价开仓(buy/sell side) →
  记录db_positions + 冷却300秒 →
  推送开仓告警
```

**保证金：** 2 USDT × 10杠杆 = 每仓位名义价值约 20 USDT

### 5.2 止盈止损逻辑

**固定止损（始终生效）：**
- 亏损 ≥ 9% → 市价平仓 → 推送告警 → 冷却300秒

**移动止盈（盈利 ≥ 4% 后激活）：**
- 激活后记录峰值（peak_price）
- 从峰值回撤 3% → 市价止盈 → 推送告警 → 冷却300秒

### 5.3 手动平仓检测（每5秒）

```python
get_position_size(symbol) → 真实持仓量
→ 如果记录中有持仓但真实持仓=0 → 判定手动平仓
→ 移除记录 + 启动冷却300秒
```
