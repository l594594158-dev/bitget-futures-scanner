## 五、跟踪止损详解

**逻辑：让利润奔跑，亏损时及时止损**

```
持仓盈利 < 5%     → 固定止损（止损线 = 成本价 × (1-止损比例)）
持仓盈利 ≥ 5%    → 止损线 = 最高价 × 97%
持仓盈利 ≥ 8%    → 止损线 = 最高价 × 95%（更宽松）
```

**举例（TSLA，止损8%）：**
- 买入价 $200，固定止损 = $200 × 0.92 = $184
- 涨到 $216（盈利8%），跟踪止损 = $216 × 0.95 = $205.2
- 继续涨到 $230，跟踪止损 = $230 × 0.95 = $218.5
- 价格跌回 $218.5，触发止损，盈利 = ($218.5-$200)/$200 = **+9.25%**
