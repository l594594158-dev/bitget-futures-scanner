## 二、餐饮行业

### 2.1 小餐饮（面馆、快餐店、奶茶店）

#### 痛点分析
- 高峰期电话接不过来，漏单严重
- 差评不知道怎么处理，或者处理完没有后续
- 不知道顾客是谁，复购率低
- 点评平台（大众点评/美团）评价懒得回
- 活动文案不会写，发朋友圈不知道发什么

#### AI 能做什么（具体场景）

**① 微信自动接单（效果：减少80%漏单）**
- 场景：顾客发微信"我要点一份牛肉面加辣"
- OpenClaw 自动识别：菜品、数量、特殊要求
- 自动回复："好的，牛肉面加辣一份，30元，预计15分钟好。取餐码62号。"
- 店家手机上看到完整订单，不用手写/手记

实现方式：
```
顾客发消息 → OpenClaw 接收 → 识别菜品/数量 → 发送确认给顾客 → 同时通知后厨
```

**② 差评处理自动化（效果：挽回30%差评顾客）**
- 场景：顾客发"太难吃了"或差评触发关键词
- OpenClaw 第一时间回复："抱歉让您失望了，方便告诉我是哪方面问题吗？给您重新做或者退款都行。"
- 记录差评内容，店家事后处理

实现方式：
```
配置关键词触发规则（难吃、不行、差、投诉等）
→ 触发后自动发送安抚话术
→ 同时记录到日志，店家通知
```

**③ 点评平台差评自动回复（效果：节省90%人力）**
- 抓取大众点评/美团的新评价
- AI 分析评价内容（好评/差评/有图）
- 差评自动生成回复："感谢您的反馈，我们非常重视，已联系您处理～"
- 好评自动回复："谢谢您的认可，欢迎下次再来！"

实现方式：
```
用 OpenClaw 网页抓取定时爬取评价页面
→ AI 生成针对性回复
→ 手动确认后一键发送（或自动发送）
```

**④ 会员唤醒与复购（效果：提升复购率15-25%）**
- 场景：顾客15天没来，自动发消息
- "张哥，您上次吃的红烧牛肉面好像挺喜欢的，最近出了新品牛杂面，要试试吗？给您留一份？"
- 节假日自动发祝福+优惠券

实现方式：
```
维护一个顾客数据库（Excel或简单表格）
→ 配置定时任务：X天未消费自动触发
→ AI 生成个性化召回话术
→ 微信自动发送
```

**⑤ 朋友圈/群运营（效果：提升曝光和到店）**
- AI 每天生成3条朋友圈文案（早中晚不同时段）
- 模板：产品图+文案+互动话题
- 店家只需复制粘贴或一键发送

实现方式：
```
输入今日食材/新品/活动 → OpenClaw 生成多条文案供选择
→ 配置定时发送 → 自动发到群/朋友圈
```

**落地步骤（1-2周完成）：**
1. Day 1：配置 OpenClaw 微信接入，测试自动回复
2. Day 3：配置关键词触发规则（接单、投诉、退款等）
3. Day 5：导入顾客名单到记忆系统
4. Day 7：配置定时任务（复购提醒、节假日问候）
5. Day 10：配置点评平台抓取+自动回复
6. Day 14：上线朋友圈文案自动生成

**预算：** 0元（用 OpenClaw 现有功能）

## 三、零售行业

### 3.1 夫妻店 / 便利店

#### 痛点分析
- 不知道怎么进货，都是凭经验
- 坐等顾客上门，没有主动营销能力
- 不知道怎么发朋友圈/群
- 顾客问有没有某商品，不知道怎么快速查

#### AI 能做什么

**⑩ 微信自动卖货（效果：线下转线上）**
- 顾客发"有没有农夫山泉"
- OpenClaw 立即回复："有，2元/瓶，整箱18元，要几瓶？"
- 顾客确认 → 记录订单 → 约定取货时间
- 整个过程不需要店家一直盯着手机

实现方式：
```
维护一个商品库存表（品名+价格+库存量）
→ 顾客发消息 → AI 匹配商品 → 确认订单 → 通知店家
```

**⑪ 促销清仓自动发圈（效果：提升库存周转）**
- 临期商品/清仓商品
- AI 生成朋友圈文案："今天矿泉水买一送一，明天过期了，邻居们有需要的快来！"
- 配合图片，一键发送

**⑫ 顾客建档（效果：知道谁是你的常客）**
- 每次顾客来买过什么，记录到记忆系统
- 下次来的时候 AI 知道他的偏好："张叔，还是老样子拿两盒烟？"
- 熟客生日自动发祝福

实现方式：
```
用 OpenClaw 记忆功能记录顾客信息
格式：姓名 | 常用商品 | 购买频率 | 生日 | 备注
→ 每次交易后更新记录
→ 触发条件自动调用
```
