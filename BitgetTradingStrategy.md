# 📊 第一部分：指标说明

## 核心指标定义

| 指标 | 参数 | 用途 |
|-----|------|-----|
| **MA** | 50/200日均线 | 趋势判断 |
| **EMA** | 9/21日指数均线 | 快速信号 |
| **RSI** | 14周期 | 超买超卖 |
| **MACD** | 12/26/9 | 动量转折 |
| **ATR** | 14周期 | 波动率/止损 |
| **Bollinger** | 20日/2σ | 突破信号 |

# 🥇 第二部分：贵金属策略

## XAUUSD (黄金)

### 指标参数
```
趋势指标：MA(50) > MA(200) → 多头排列
快速指标：EMA(9) > EMA(21) → 看涨
动量指标：MACD(12,26,9) 信号线交叉
超买超卖：RSI(14) < 30 超卖，> 70 超买
波动率：ATR(14) 突破20日高点
```

### 入场信号（做多）
```
必须同时满足：
✅ MA(50) > MA(200)（趋势多头）
✅ 价格回踩 MA(50) 或 EMA(9)
✅ RSI(14) 从 <40 区域回升
✅ MACD 信号线从下方穿越 MACD 线
✅ 成交量放大（若有）

或 突破入场：
✅ 价格突破 20日高点 + RSI(14) > 60
```

### 入场信号（做空）
```
必须同时满足：
✅ MA(50) < MA(200)（趋势空头）
✅ 价格反弹至 MA(50) 或 EMA(9)
✅ RSI(14) 从 >60 区域回落
✅ MACD 信号线从上方穿越 MACD 线
```

### 出场规则
```
止盈：
  - RSI(14) > 75 → 减半仓
  - RSI(14) > 80 → 全出
  - 固定目标：ATR(14) × 3

止损：
  - 固定止损：入场价 × 3% 止损
  - 跟踪止损：最高点回落 ATR(14) × 2

出场：
  - 跌破 MA(50) 且 MA(50) 转跌
  - MACD 死叉
```

### 仓位管理
```
仓位上限：总资金 5%（10倍杠杆）
最小波动：ATR(14) × 2 < 止损幅度
加仓规则：突破后回踩不破前低可加仓一次
```

## 通用入场信号（美股）

### 做多入场
```
1. 趋势确认：
   MA(50) > MA(200) ✓

2. 回调入场（首选）：
   价格回撤至 MA(50) 附近（±2%）
   RSI(14) 在 40-50 区间止跌
   缩量调整，放量反弹

3. 突破入场（次选）：
   价格突破 20日高点
   成交量 > 20日均量 1.5倍
   RSI(14) > 55

4. 动量确认：
   MACD(12,26,9) 在零轴上方
   EMA(9) > EMA(21)
```

### 做空入场
```
1. 趋势确认：
   MA(50) < MA(200) ✓

2. 反弹入场：
   价格反弹至 MA(50) 附近（±2%）
   RSI(14) 在 50-60 区间滞涨
   放量下跌

3. 动量确认：
   MACD(12,26,9) 在零轴下方
```

---

## 出场规则（美股）

### 止盈
```
分批止盈：
  - 涨幅达到 ATR × 4 → 减仓 1/3
  - 涨幅达到 ATR × 6 → 再减 1/3
  - 剩余持有至跟踪止损

RSI 极端值：
  - RSI(14) > 80 → 全部止盈
  - RSI(14) > 75 → 减半仓
```

### 止损
```
固定止损：按等级表格执行
跟踪止损：
  - 最高点回落 ATR × 2 全出
  - 或回落 3% 全出

时间止损：
  - 入场 5 日未盈利且趋势走坏 → 出局
```

# 🛡️ 第七部分：风控核心规则

## 仓位管理

```
总仓位上限：不超过资金 80%
保留现金：至少 20%（应对极端行情）

单笔仓位：
  A+级：2%
  A级：3%
  B级：4%
  C级：5%
  贵金属：4-5%
  杠杆ETF：单只 ≤ 10%
```

## 强制止损表

| 资产类别 | 最大止损 | 跟踪止损触发 |
|---------|---------|------------|
| 贵金属 | 3-4% | ATR × 2 |
| 美股A级 | 6-8% | ATR × 2.5 |
| 美股B/C级 | 4-5% | ATR × 2 |
| 商品ETF | 4-5% | ATR × 2 |
| TQQQ/SQQQ | 10% | ATR × 3 |
| SGOV | 无需止损 | - |

## 日级风控

```
单日最大亏损：3%
  → 触发后：关闭所有仓位，禁止新开单

单日最大盈利：不做限制

连续亏损：
  - 亏损 3 次 → 仓位减半
  - 亏损 5 次 → 当日停止交易
```

## 月度风控

```
月度最大亏损：10%
  → 触发后：进入 3 天冷静期，只做观望

月度盈利目标：不做设定（让利润奔跑）
```

## 故障保护

```
断线保护：
  每 5 分钟检查 API 连接
  断开时自动取消所有挂单

恢复机制：
  重连后先同步持仓
  确认无误后再恢复交易
  记录故障日志
```

# 🔄 第八部分：策略执行流程

## 每日操作时间轴

| 时间 | 操作 |
|-----|-----|
| **盘前 30 分钟** | 检查隔夜持仓，查看期货/外汇预判 |
| **盘中每小时** | 扫描标的是否触发信号 |
| **盘后 15 分钟** | 复盘当日交易，更新持仓 |
| **每日收盘** | 执行止损/止盈单 |

## 信号扫描优先级

```
1. 检查止损单是否触发
2. 检查止盈单是否触发
3. 扫描新的入场信号
4. 检查是否需要加仓
5. 更新风控状态
```

# ⚠️ 第十部分：重要提醒

## 必须遵守的铁律

```
1. ❌ 不要逆势加仓
2. ❌ 不要在低波动时追突破（假突破多）
3. ❌ 不要重仓单一标的（上限 5%）
4. ❌ 不要忽略止损（止损是生命线）
5. ❌ 不要在 RSI 极端时入场
6. ❌杠杆ETF不要持仓过周末（除非明确趋势）

✅ 轻仓试错，盈利加码
✅ 截断亏损，让利润奔跑
✅ 定期复盘，优化参数
✅ 保留现金，等待机会
```

## 策略不适用的行情

```
1. 消息面主导的剧烈波动（如非农、FOMC）
   → 策略：提前减仓或出局
   
2. 假期前后低流动性
   → 策略：缩小仓位
   
3. 重大黑天鹅事件
   → 策略：止损可能无法成交，保命优先
   
4. 盘整震荡市
   → 策略：减少操作，等突破确认
```
