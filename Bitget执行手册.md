## 3.2 止盈规则

### RSI 极端值止盈
```
RSI > 75  → 减仓 1/3
RSI > 80  → 再减 1/3
RSI > 85  → 清仓
```

### 分批止盈（按 ATR 计算）
```
目标1：ATR × 4  → 减仓 1/3
目标2：ATR × 6  → 再减 1/3
目标3：ATR × 8  → 全部止盈
```

### 趋势破坏止盈（移动止损）
```
□ 跌破 MA(50) 且均线拐头 → 清仓
□ MACD 死叉 → 清仓
□ 最高点回落 5% → 清仓
```

# 📌 第四章：仓位计算

## 4.1 仓位公式

```
仓位(数量) = 账户净值 × 仓位比例% ÷ (当前价格 × 杠杆倍数)
```

## 4.2 各标的仓位上限

```
贵金属：5%（10倍杠杆）
美股A+级：2%（5倍杠杆）
美股A级：3%（5倍杠杆）
美股B/C级：4-5%（3-5倍杠杆）
中概股：2%（5倍杠杆）
商品ETF：4%（5倍杠杆）
TQQQ/SQQQ：10%（3倍杠杆）
VTI：5%（3倍杠杆）
SGOV：20%（1倍，允许大仓位停泊）
```

## 4.3 加仓规则

```
✅ 只在盈利时加仓
✅ 回调不破前低才能加仓
✅ 同一标的最多加仓 1 次
✅ 加仓后立即调整止损至保本
```

# 📌 第五章：风控红线

## 5.1 强制止损线

```
🟢 正常状态：单笔亏损 2% 以内
🟡 警戒状态：单日亏损 3% → 禁止新开单
🔴 熔断状态：月度亏损 10% → 3天冷静期
```

## 5.2 连续亏损处理

```
亏损 2 次 → 检查策略是否失效
亏损 3 次 → 仓位减半
亏损 5 次 → 当日停止交易
```

## 5.3 禁止交易条件

```
□ 重大消息面（非农 / FOMC / CPI）
□ 开盘前 30 分钟不做单
□ 低流动性时段（假期前后）
□ 策略信号不明确时
□ 情绪化状态下
```

# 📌 第六章：每日操作流程

## 盘前（09:00-09:30）

```
Step 1: 查看期货/贵金属预判今日方向
Step 2: 检查隔夜持仓状态
Step 3: 计算今日可承受亏损额度
Step 4: 列出今日关注标的清单
```

## 盘中（09:30-16:00）

```
每30分钟扫描一次：
┌────────────────────────────────────┐
│ 1. 持仓是否触发止损？                │
│ 2. 持仓是否触发止盈？                │
│ 3. 有无新标的出现入场信号？          │
│ 4. 需不需要加仓？                    │
│ 5. 风控状态是否正常？                │
└────────────────────────────────────┘
```

## 盘后（16:00-16:30）

```
Step 1: 复盘今日所有交易
Step 2: 记录盈亏和原因
Step 3: 更新持仓清单
Step 4: 准备明日交易计划
Step 5: 执行未成交的挂单
```
