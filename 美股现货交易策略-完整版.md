# 一、交易原则

## 1.1 核心铁律

```
✅ 只做现货，不做合约
✅ 只用闲钱，不杠杆
✅ 止损优先，截断亏损
✅ 让利润奔跑，及时止盈
✅ 顺势而为，不逆势
```

## 1.2 仓位原则

| 情况 | 仓位上限 |
|-----|---------|
| 单笔交易 | 总资金 20% |
| 同时持仓 | 总资金 60% |
| 止损幅度 | 单笔 -5% |
| 日最大亏损 | 总资金 3% |

# 二、技术分析指标

## 2.1 趋势指标

### 均线系统 (MA)

| 均线 | 周期 | 用途 |
|-----|------|------|
| MA5 | 5日 | 短期趋势 |
| MA20 | 20日 | 中期趋势 |
| MA60 | 60日 | 长期趋势 |
| MA200 | 200日 | 牛熊分界线 |

**信号判断：**

| 状态 | 条件 | 信号 |
|-----|------|------|
| 多头排列 | MA5 > MA20 > MA60 | 🟢 强烈看涨 |
| 空头排列 | MA5 < MA20 < MA60 | 🔴 强烈看跌 |
| 均线纠缠 | MA5 MA20 MA60 混乱 | 🟡 震荡市 |

### 趋势线

- 画法：连接两个低点/高点
- 突破趋势线：可能反转
- 角度过陡：不可持续

---

## 2.2 动量指标

### RSI（相对强弱指数）

| RSI范围 | 信号 | 含义 |
|--------|------|------|
| 0-30 | 🔴 超卖 | 可能反弹 |
| 30-50 | 🟡 偏弱 | 观望 |
| 50-70 | 🟡 偏强 | 可持有 |
| 70-100 | 🔴 超买 | 可能回调 |

**特殊用法：**
- RSI 创6个月新低 → 强烈买入信号
- RSI 创6个月新高 → 强烈卖出信号
- 股价创新高但RSI未创新高 → 顶背离（卖出信号）
- 股价创新低但RSI未创新低 → 底背离（买入信号）

### MACD

| 状态 | 信号 |
|-----|------|
| DIF > DEA 且 MACD柱 > 0 | 🟢 金叉（买） |
| DIF < DEA 且 MACD柱 < 0 | 🔴 死叉（卖） |
| MACD柱由负转正 | 🟢 买入信号 |
| MACD柱由正转负 | 🔴 卖出信号 |

**背离信号：**
- 顶背离：价格创新高但MACD没创新高 → 卖出
- 底背离：价格创新低但MACD没创新低 → 买入

---

## 2.3 波动率指标

### 布林带 (Bollinger Bands)

| 轨道 | 含义 |
|-----|------|
| 上轨 +2σ | 压力位（超买区域）|
| 中轨 20日MA | 中性 |
| 下轨 -2σ | 支撑位（超卖区域）|

**交易信号：**

| 价格位置 | 信号 | 行动 |
|---------|------|------|
| 触及下轨 | 🔴 超卖 | 关注买入机会 |
| 触及上轨 | 🔴 超买 | 关注卖出机会 |
| 突破上轨 | 🟢 强势 | 可能继续涨 |
| 跌破下轨 | 🔴 弱势 | 可能继续跌 |

**布林带收口**：波动率降低，可能有大行情

### ATR（真实波幅）

| 用途 | 说明 |
|-----|------|
| 止损设置 | 止损 = 入场价 - 2×ATR |
| 目标位 | 止盈 = 入场价 + 3×ATR |
| 仓位计算 | 仓位 = 账户×风险% ÷ (2×ATR) |

---

## 2.4 成交量指标

| 信号 | 含义 | 行动 |
|-----|------|------|
| 放量上涨 | 买盘强劲 | 🟢 顺势买入 |
| 放量下跌 | 卖盘强劲 | 🔴 顺势卖出 |
| 缩量上涨 | 可能见顶 | ⚠️ 谨慎 |
| 缩量下跌 | 可能见底 | ⚠️ 关注 |
| 放量突破 | 趋势确认 | ✅ 确认信号 |

# 三、买入分析系统

## 3.1 买入条件（必须同时满足）

| # | 条件 | 技术要求 |
|---|------|---------|
| 1 | 趋势确认 | MA20 > MA60（多头）或股价在MA20上方 |
| 2 | RSI位置 | RSI < 60（不是超买）|
| 3 | 动能确认 | MACD柱 ≥ 0 或 DIF > DEA |
| 4 | 支撑位 | 价格在关键支撑位附近 |
| 5 | 成交量 | 回调缩量，上涨放量 |

---

## 3.2 买入信号强度分级

| 强度 | 满足条件 | 仓位 | 止损 |
|-----|---------|------|------|
| 🟢 强烈 | 5个全满足 | 20% | -5% |
| 🟡 中等 | 4个满足 | 15% | -5% |
| 🟠 较弱 | 3个满足 | 10% | -4% |
| 🔴 不建议 | <3个 | 不入场 | - |

---

| 信号 | 说明 | 优先级 |
|-----|------|-------|

| MACD底背离 | 价格新低但MACD未新低 | ⭐⭐⭐ |
| 布林下轨反弹 | 价格触及下轨后出现阳线 | ⭐⭐⭐ |

|-----|------|-------|

## 3.4 买入流程

```
1. 扫描标的池（每天收盘后）
   ↓
2. 筛选出现买入信号的标的
   ↓
3. 确认信号强度
   ↓
4. 计算仓位和止损位
   ↓
5. 执行买入
   ↓
6. 设置止损单
   ↓
7. 记录交易
```

# 六、综合决策流程

## 6.1 买入决策树

```
是否有买入信号？
├── 否 → 不操作
└── 是 → 信号强度如何？
         ├── 强烈(5个条件)
         │   └── 仓位20%，止损-5%
         ├── 中等(4个条件)
         │   └── 仓位15%，止损-5%
         ├── 较弱(3个条件)
         │   └── 仓位10%，止损-4%
         └── <3个条件
             └── 不建议买入
```

## 6.2 卖出决策树

```
持仓标的是否有卖出信号？
├── 止损/紧急信号
│   └── 立即清仓
├── 警告信号
│   └── 减仓50%
├── 关注信号
│   └── 持有观察
└── 无信号
    └── 继续持有
```

## 6.3 每日检查清单

```
盘前（9:00前）：
□ 检查隔夜美股期货
□ 查看VIX指数
□ 回顾持仓
□ 查看今日重大事件

盘中（9:30-16:00）：
□ 每30分钟扫描持仓
□ 有信号及时响应
□ 监控止损/止盈

盘后（16:00后）：
□ 复盘今日交易
□ 扫描潜在标的
□ 准备明日计划
```

# 七、风险控制

## 7.1 仓位管理

| 账户规模 | 单笔上限 | 总持仓上限 |
|---------|---------|-----------|
| 小于$5000 | 20% | 60% |
| $5000-20000 | 15% | 50% |
| 大于$20000 | 10% | 40% |

## 7.2 强制止损

| 类型 | 触发条件 | 行动 |
|-----|---------|------|
| 单笔止损 | -5% | 强制卖出 |
| 日止损 | -3% | 停止当日交易 |
| 周止损 | -8% | 减少仓位 |
| 月止损 | -15% | 进入冷静期 |

## 7.3 连续亏损处理

| 连续亏损次数 | 仓位限制 |
|-------------|---------|
| 2次 | 减半仓 |
| 3次 | 再减半 |
| 5次 | 停止交易1天 |
