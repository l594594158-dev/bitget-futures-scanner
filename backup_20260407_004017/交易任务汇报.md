## 四、评分系统

### 4.1 技术分析评分（基础分）

基于 1 小时 K 线数据计算。

| 指标 | 条件 | 加分 | 减分 |
|-----|------|:----:|:----:|
| **均线 MA** | MA5 > MA20 > MA50（多头排列） | +2 | - |
| | MA5 < MA20 < MA50（空头排列） | - | +2 |
| **RSI** | < 35（超卖） | +2 | - |
| | 35-45（偏弱） | +1 | - |
| | 60-70（偏强） | - | +1 |
| | > 70（超买） | - | +2 |
| **MACD** | 金叉（柱状图由负转正） | +2 | - |
| | 死叉（柱状图由正转负） | - | +2 |
| | 柱状图扩张（正值且放大） | +1 | - |
| | 柱状图收缩（负值且缩小） | - | +1 |

### 4.2 新闻情绪评分（附加分）

| 来源 | 方式 | 分数范围 |
|-----|------|---------|
| Google News RSS | 关键词匹配情绪词 | -10 ~ +10 |

**新闻情绪词示例：**

| 正面词 | 负面词 |
|--------|--------|
| buy, bullish, upgrade, surge, beat | sell, bearish, downgrade, crash, miss |

### 4.3 综合评分计算

```
综合评分 = 技术评分 + 新闻评分(限制后)

新闻评分(限制后) = max(-1, min(1, 原始新闻分数))
```

**新闻评分权重限制为 ±1 分，避免新闻噪音干扰技术信号。**

## 五、交易规则

### 5.1 开仓条件（必须同时满足）

```
┌─────────────────────────────────────────┐
│            ✅ 开仓条件（AND）            │
├─────────────────────────────────────────┤
│  ✓ 当前无持仓                            │
│  ✓ 余额 ≥ 25 USDT                       │
│  ✓ 综合评分 ≥ 4 分                       │
│  ✓ 新闻评分(限制后) ≥ -1                 │
└─────────────────────────────────────────┘
```

**不同市场情绪下的技术门槛：**

| 新闻评分 | 技术评分需达到 |
|---------|--------------|
| +1（偏多） | ≥ 3 分 |
| 0（中性） | ≥ 4 分 |
| -1（偏空） | ≥ 5 分 |

### 5.2 平仓条件（满足任一即触发）

```
┌─────────────────────────────────────────┐
│            🔴 平仓条件（OR）             │
├─────────────────────────────────────────┤
│  ○ 综合评分 ≤ -4 分                       │
│  ○ RSI > 80（极端超买）                 │
│  ○ 重大利空事件 且 新闻评分 < -3        │
│  ○ 触发固定止损                          │
│  ○ 触发跟踪止损                          │
└─────────────────────────────────────────┘
```

### 5.3 止损规则

| 类型 | 触发条件 | 计算方式 |
|-----|---------|---------|
| **固定止损** | 买入后任何时候 | 买入价 × (1 - 止损比例) |
| **跟踪止损** | 盈利 ≥ 5% | 最高价 × 97% |
| **跟踪止损** | 盈利 ≥ 8% | 最高价 × 95% |

**止损比例：**

| 标的类型 | 止损比例 |
|---------|---------|
| 高波动（TSLA、NVDA） | 8% |
| 中波动（其他） | 5-6% |
