## 下午补充（17:00-19:24）

### API接入关键发现
- 之前用 B-API-KEY header 失败150+次
- 正确方法是使用 ACCESS-KEY header
- 签名：HMAC-SHA256，消息格式：timestamp + method + path
- Header格式：
  - ACCESS-KEY（不是B-API-KEY）
  - ACCESS-SIGN（不是B-SIGNATURE）
  - ACCESS-TIMESTAMP
  - ACCESS-PASSPHRASE
